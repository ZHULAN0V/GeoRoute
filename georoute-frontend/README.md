# Frontend на React

React‑приложение для работы с GPX‑файлами: загрузка, парсинг маршрутов (`<trk>`) и маркеров (`<wpt>`), отображение на интерактивной карте, управление через Redux.

## Стек технологий

* React;
* TypeScript;
* Redux Toolkit;
* Vite;
* Leaflet / Google Maps API;
* Axios;
* Material UI

## Установка и запуск

**Требования:** Node.js 18+, npm 8+.

**Установка зависимостей:**

```bash
npm install
```

**Команды для работы:**

```bash
# запуск в режиме разработки
npm run dev
# сборка проекта
npm run build
# запуск сбилженой версии
npm run preview
```

## Ограничения

* Привзка длинного маршрута к тропам на карте можеть длиться до 1 минуты
* Маршрут с большим колличеством точек сильно нагружает браузер, что вызывает проблемы при взаимодействии с маршрутом
